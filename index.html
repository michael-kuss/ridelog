ridelog

This is the repository for the ridelog app. We try to build an easy to use interface for all who want to track their tours and exchange their tour expierence with their friends.

Plans

Android App

  • Livetracking from an android app for others to see live from the internet -- HTTP transfer of gps data and resendending missed data if dataconnection is lost
  • find friends that are nearby
  • Bett&Bike/archies Europe integration?!? (don't know yet)
  • import from Garmin GPS (gpsmap 62/Edge 705)
  • exchange imported GPS data with livetracking data?!?

Website

  • Report on tours with daily route, pictures, ...
  • Information shown on map should also provide additional information such as speed, temp, cadence, ...
  • monthly, weekly, daily reports
